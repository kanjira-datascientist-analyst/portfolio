Severe Acute Malnutrition (SAM) Risk Predictor

This project shows how Machine Learning (ML) can support early detection of severe acute malnutrition (SAM). Using an XGBoost model trained on synthetic pediatric data from Hugging Face, I built a web application that flags high-risk cases needing urgent intervention and distinguishes them from those suitable for routine monitoring. The system uncovers clinical patterns in malnutrition progression, predicts severe outcomes, and delivers evidence-based, personalized recommendations, demonstrating how predictive analytics can improve malnutrition management in resource-limited settings.

Tableau project

This project analyzes global skill migration patterns from 2015 to 2019 using data sourced from Kaggle.com, uncovering countries experiencing brain drain versus brain gain across 142 nations. It reveals key trends in the movement of highly skilled workers, identifies emerging professions shaping the global workforce, and showcases regional migration dynamics. With interactive filters and dimensions, the dashboard offers valuable insights for understanding how talent flows impact economies and labor markets worldwide.
